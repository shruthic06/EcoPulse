# 🌿 EcoPulse — AI-Powered Sustainable Fashion Checker

EcoPulse helps shoppers make informed, sustainable fashion choices. It consists of a Chrome extension that extracts product data from retailer pages, a Python AI service that analyzes sustainability using OpenAI, and a web dashboard that displays the results.

## How It Works

1. Browse any fashion retailer (SHEIN, H&M, Zara, etc.)
2. Click the EcoPulse extension widget — it auto-extracts product name, brand, fabric composition, sustainability claims, and certifications
3. Hit Analyze — the extension sends the data to the backend, which calls OpenAI for a real sustainability analysis
4. Click "Open Full Dashboard" to see the full AI-powered analysis on the web dashboard
5. Earn reward points for sustainable actions like donating, repairing, and rewearing clothes

## Project Structure

```
packages/
├── ai-service/       # Python FastAPI service — calls OpenAI for analysis & chat
├── backend/          # Node.js Express server — API + dashboard
├── extension/        # Chrome extension — content script + popup widget
├── shared/           # Shared TypeScript types
└── web-app/          # Web app rendering utilities
```

## Tech Stack

- **Extension**: Vanilla JS content script injected into retailer pages
- **Backend**: Node.js + Express + TypeScript
- **AI Service**: Python + FastAPI + OpenAI API (GPT-4o)
- **Dashboard**: Single-page HTML served by the backend
- **Testing**: Vitest

## Prerequisites

- Node.js 20+
- Python 3.11+
- An OpenAI API key

## Setup

### 1. Install Node dependencies

```bash
npm install
npm run build
```

### 2. Set up the Python AI service

```bash
cd packages/ai-service
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Configure your OpenAI API key

```bash
cp packages/ai-service/.env.example packages/ai-service/.env
```

Edit `packages/ai-service/.env` and replace `sk-your-key-here` with your actual OpenAI API key.

## Running

You need two terminals:

**Terminal 1 — Python AI service:**

```bash
cd packages/ai-service
source venv/bin/activate
uvicorn main:app --reload --port 8000
```

**Terminal 2 — Node backend:**

```bash
node packages/backend/dist/start.js
```

The dashboard is available at `http://localhost:3000`.

## Loading the Chrome Extension

1. Open `chrome://extensions` in Chrome
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `packages/extension` folder
5. The EcoPulse widget (🌿) will appear on retailer product pages

## Features

### Chrome Extension
- Auto-extracts product name, brand, fabric composition, sustainability claims, and certifications from retailer pages
- Works on SHEIN, H&M, Zara, ASOS, Nike, Amazon, and most fashion retailers
- Shows quick sustainability scores in the popup
- "Open Full Dashboard" passes all data to the web dashboard for detailed AI analysis

### AI-Powered Analysis (via OpenAI)
- Per-material CO₂ and water usage estimates
- Health risk assessment and chemical risk identification
- Greenwashing detection — flags vague claims not backed by certifications
- Biodegradability estimation
- Overall impact score with environmental, health, and greenwashing sub-scores
- No hardcoded data — all analysis is generated by GPT-4o

### Web Dashboard
- Overall impact score ring with color-coded rating
- Category cards: Environmental Impact, Health Transparency, Greenwashing Risk, Circularity Score
- Biodegradability timeline with visual indicator
- "What does this mean?" AI-generated explanation
- Better Choices section with sustainable alternatives
- AI chatbot (green floating bubble) for asking questions about the product
- Loading screen while AI analyzes — no flash of static content

### Sell Clothes
- Upload photos and list items for sale
- Uploaded images appear in My Listings
- Listing an item earns 25 reward points

### Rewards
- Earn points for sustainable actions: donations (+25), repairs (+10), rewears (+5), recycling (+20)
- Points tracked per session with level progression (Seedling → Sprout → Tree → Earth Guardian)
- Rewards dropdown in the top bar for quick access

## API Endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/` | GET | Serves the dashboard |
| `/api/analyze` | POST | Legacy analysis (local scoring) |
| `/api/ai-analyze` | POST | AI-powered analysis via OpenAI |
| `/api/chat` | POST | AI chatbot (proxied to Python service) |
| `/api/alternatives` | GET | Query sustainable alternatives |
| `/api/rewards` | POST | Log a reward event |
| `/api/rewards/:userId` | GET | Get balance and history |
| `/api/locations` | GET | Find nearby donation/recycling centers |
| `/images/:filename` | GET | Serve static product images |

## Environment Variables

| Variable | Location | Description |
|---|---|---|
| `OPENAI_API_KEY` | `packages/ai-service/.env` | Your OpenAI API key (required) |

| `AI_SERVICE_URL` | Node backend env | Python service URL (default: `http://localhost:8000`) |
| `PORT` | Node backend env | Dashboard port (default: `3000`) |
